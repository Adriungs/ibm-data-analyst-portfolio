# 🚀 Data Analyst: Career Preparation - Cognitive Class

**Mi journey completo** en la carrera de **Data Analyst** de Cognitive Class (IBM).

Estoy documentando todo mi aprendizaje paso a paso para construir un portafolio real, demostrar las habilidades adquiridas y tener un registro claro de mi progreso.

### 📌 Sobre el programa
- **Nombre del Learning Path**: Data Analyst: Career Preparation
- **Contenido**: 2 cursos + 5 Guided Projects
- **Objetivo**: Aprender el stack completo de análisis de datos (Python, SQL, Pandas, Tableau) mediante proyectos prácticos y construir un portafolio profesional.

**Skills que voy a adquirir**:
- Python para Data Science
- Pandas, NumPy y scikit-learn
- SQL y bases de datos relacionales
- Data Wrangling y limpieza de datos
- Exploratory Data Analysis (EDA)
- Visualización de datos con Tableau
- Creación de dashboards
